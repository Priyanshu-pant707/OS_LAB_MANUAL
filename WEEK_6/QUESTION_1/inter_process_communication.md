# Interprocess communication 
- it contains three most important concept 
   1. PIPE
   2. Message Passing 
   3. Shared Memory

- helps in data exhange between the parent and child process ( or 2 process) 



## 1. ***PIPE**
- it is a unidirectional communication channel 
- data flow can be like  : child->parent or parent -> child.
- it is like a temporary memory buffr inside the operating system
